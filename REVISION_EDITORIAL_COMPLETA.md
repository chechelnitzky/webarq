# Revisión editorial completa

Se reescribieron los textos principales, guías y preguntas frecuentes para que la web aporte valor real, no solo frases generales.

Cambios principales:
- Guías con rangos en UF/m² para construcción y remodelación.
- Honorarios profesionales publicados solo en rangos referenciales validados: diagnóstico, anteproyecto, proyecto, coordinación y acompañamiento.
- Preguntas frecuentes con respuestas completas y útiles.
- Páginas de permisos, regularización, recepción final, ampliación, remodelación, oficinas y locales con tablas, checklists, errores frecuentes y criterios prácticos.
- Páginas por comuna con contenido menos genérico y más contextual.
- CSS agregado para tablas editoriales, listas de chequeo y mejor lectura móvil.
